Amicus

An experimental RAG tutor that explains programming, statistics, and NLP through legal analogies. Built as a teaching demo for how domain-anchored pedagogy and retrieval grounding make technical topics more accessible to learners with a legal background.

Tech Stack

  • LLM: gemini-3-flash-preview
  • Embeddings: Gemini Embedding 001
  • Vector Store: ChromaDB
  • Framework: LangChain
  • Search: Hybrid (Vector + BM25)
  • Agent: ReAct (4 topic tools)
  • Frontend: Vanilla JS + FastAPI
Architecture 
Data Sources (.docx, .pdf, .ipynb, .py, .md)
  → Multi-Format Loader
  → Chunking (500 chars / 100 overlap)
  → Embedding (Gemini Embedding 001)
  → ChromaDB Vector Store
  → Hybrid Retrieval (Vector + BM25)
  → Routing (Authority / Agent / RAG)
  → ReAct Agent (4 topic tools)
  → gemini-3-flash-preview
  → Pedagogical Response + Sources
